### 前言
这是一个搭建webpack+react项目脚手架的简易教程，参照了[create-react-app](https://github.com/facebookincubator/create-react-app)的配置方式。每一个例子都有非常详细的搭建步骤。

### 运行并启动项目

克隆项目到本地以后，选择要运行的例子进入到该项目路径下

安装：`npm install`

运行：`npm start`
